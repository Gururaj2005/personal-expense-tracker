# Personal Expense Tracker

A Python-based personal expense tracker with a graphical user interface (GUI) built using Tkinter.

## Features
- Add and manage income and expenses
- Automatic balance calculation
- Data stored in JSON file
- Low balance alert system
- Email notifications
- SMS notifications using Twilio

## Technologies Used
- Python
- Tkinter (GUI)
- JSON (data storage)
- SMTP (Email alerts)
- Twilio API (SMS alerts)

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install python-dotenv twilio
3. Add your credentials in `.env` file
4. Run the program:
   python test (1).py

## Note
Do not upload your `.env` file. Keep your credentials safe.
